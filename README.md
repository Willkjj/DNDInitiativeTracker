# DNDInitiativeTracker
A simple personal project for DMs to keep track of Initiative and other basic combat features.


<a target="_blank" href="https://icons8.com/icon/104704/dungeons-and-dragons">Dungeons and Dragons</a> icon by <a target="_blank" href="https://icons8.com">Icons8</a>

# Project Setup
Running this project involves vite and websocket. To set up these modules, run 
`npm install express socket.io vite`
And
`npm run dev`
